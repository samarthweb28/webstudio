Who's building this

I'm Samarth, the developer behind Samarth Web Studio. I build websites with plain HTML, CSS and JavaScript — no unnecessary frameworks, just fast-loading pages that look intentional on every screen size.

My process is simple: understand what the site needs to say, design a layout that says it clearly, then write clean code that's easy to maintain long after launch.
